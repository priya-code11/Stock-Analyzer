About Stock AI Analyzer

Smart decision-making using real-time data & AI

📊 Project Overview

The Stock AI Analyzer is a modern web application built using Django that helps users make intelligent trading decisions. It fetches real-time market data and applies technical analysis to generate actionable insights like BUY, SELL, or HOLD.

🚀 Features

  • 📡 Real-time stock data using API
  • 🤖 AI-based prediction system
  • 📉 Moving Average (MA20) analysis
  • 📊 RSI (Relative Strength Index)
  • 🌙 Modern dark + glass UI
  • 📁 Portfolio tracking (future scope)

⚙️ Technologies Used

  • 🐍 Python (Django Framework)
  • 🌐 HTML, CSS, Bootstrap
  • 📊 yFinance API for stock data
  • 📦 Pandas for data processing

🧠 How It Works

The system fetches historical stock data and calculates key indicators like Moving Average (MA20) and RSI.

  • BUY → Price above average & strong trend
  • SELL → Weak trend or falling market
  • HOLD → Market uncertainty
