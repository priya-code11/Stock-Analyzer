About Our Project

Smart decision-making using real-time data and AI logic

📊 Project Overview

The Stock AI Analyzer is a web-based application built using Django that helps users make better trading decisions. It uses real-time stock data and applies technical indicators to predict whether a stock should be BUY, SELL, or HOLD.

🚀 Features

  • Real-time stock data using API
  • AI-based prediction system
  • Moving Average (MA20) analysis
  • RSI (Relative Strength Index)
  • Clean and modern dark UI
  • Portfolio tracking (future scope)

⚙️ Technologies Used

  • Python (Django Framework)
  • HTML, CSS, Bootstrap
  • yFinance API for stock data
  • Pandas for data analysis

🧠 How It Works

The system fetches historical stock data and calculates key indicators like Moving Average and RSI. Based on these values:

  • BUY → Price above average & strong trend
  • SELL → Weak trend or falling market
  • HOLD → Uncertain conditions

🔮 Future Enhancements

  • Machine Learning model (LSTM)
  • Live candlestick charts
  • Auto-refresh real-time data
  • User login & portfolio saving
