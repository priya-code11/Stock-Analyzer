About Stock AI Analyzer

Smart decisions using data & AI

📊 Project Overview

Stock AI Analyzer is a modern web app built with Django that helps users make smarter trading decisions using real-time data and technical analysis. It generates signals like BUY, SELL, and HOLD.

🚀 Features
  • 📡 Real-time stock data
  • 🤖 AI-based prediction
  • 📉 Moving Average (MA20)
  • 📊 RSI indicator
  • 🌙 Clean dark UI
  • 📁 Portfolio tracking (coming soon)
⚙️ Technologies
  • 🐍 Django (Python)
  • 🌐 HTML, CSS, Bootstrap
  • 📊 yFinance API
  • 📦 Pandas
🧠 How It Works

The system analyzes stock data using indicators like Moving Average and RSI to determine market conditions.

  • BUY → Strong trend
  • SELL → Weak or falling trend
  • HOLD → Uncertain market
